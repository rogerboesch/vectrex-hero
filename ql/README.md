# R.E.S.C.U.E. -- Sinclair QL

Pixel-based platformer targeting the Sinclair QL with its Motorola 68008 CPU. Runs as a native QDOS executable.

## Build

```bash
cd src
make        # Compile and convert to QDOS format
make clean  # Remove build artifacts
```

Output: `rescue` (QDOS executable with self-relocating loader)

### Toolchain

The build pipeline chains several tools:

```
C source --> vbcc (68K compiler) --> vasm (assembler) --> vlink (linker, TOS format)
    --> tos2ql (custom converter) --> QDOS executable
```

- **VBCC**: C cross-compiler at `/Users/roger/retro-tools/atarist-sdk/vbcc/`
- **VASM**: 68K assembler at `/Users/roger/retro-tools/vasm/`
- **tos2ql**: Custom tool (`tools/ql/tos2ql/`) that converts Atari TOS executables to self-relocating QDOS binaries with XTcc trailer

## Architecture

### Hardware

- **CPU**: Motorola 68008 @ 7.5MHz (8-bit data bus, 20-bit address)
- **Display**: 256x256 pixels, Mode 8 (8 colors)
- **RAM**: 128KB (QL standard)

### Rendering

Pixel-based rendering with 4bpp nibble-packed sprite data:
- Room-based scrolling (not tile-based)
- Sprites have variable dimensions (width, height, pixel data pointer)
- Color indices 0-7 matching QL Mode 8 palette
- Assembly routines for screen addressing and color operations (`ql_hw.s`)

### Physics

All speeds scaled ~3.5x from the original design to account for the QL's ~17fps frame rate.

### Level System

Room-based layout shared with the Vectrex version via `../vectrex/src/levels.h`. Each room has walls, enemies, and exits connecting rooms.

## Source Files

| File | Description |
|------|-------------|
| `main.c` | Game loop, state machine |
| `player.c` | Player physics (68K-scaled timing) |
| `enemies.c` | Enemy AI |
| `render.c` | Pixel rendering (largest module, ~38KB) |
| `levels.c` | Level/room data |
| `sprites.c` | Sprite data (generated by QL Workbench) |
| `ql_hw.s` | Assembly hardware routines |
| `crt0.s` | Runtime startup code |

## Development Tool

**QL Workbench** (`tools/ql/workbench/`) provides:
- Sprite editor with QL palette
- Built-in iQL emulator (full 68008 + QDOS emulation)
- M68K disassembler and debugger
- Breakpoints and memory inspector
- Export sprites to C source
